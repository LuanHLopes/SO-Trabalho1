- Para melhor funcionamento dos requisitos java, executar o comando java -jar filepath.jar no cmd em modo administrador.

- Nesta pasta constam 3 projetos: ProducerConsumer_WaitNotify que é um simulador local, utilizando implementação Wait/Notify; ProducerJgroups e ConsumerJGroups que é um simulador utilizando passagem de mensagem (Message Passing), pode ser rodado localmente ou em rede.

- Para o simulador do Producer/Consumer Message Passing é necessário que ambos os dispositivos estejam na mesma rede.

- Iniciar primeiro o produtor (ProducerJGroups.jar) e logo após o consumidor (ConsumerJGroups.jar).
